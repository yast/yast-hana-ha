Watchdog Configuration

A watchdog device is required to be loaded in order for the SBD to work correctly.

If your hardware or virtualization platform does not offer a watchdog, make sure that you at least configure the software watchdog, "softdog", module.

Watchdogs to Configure
is a list of watchdog modules to be configured to load at the system boot (by listing them in /etc/modules-load.d/yast.conf).
Configured Watchdogs
shows a list of watchdog already added to /etc/modules-load.d/yast.conf).
Loaded watchdogs
shows a list of already loaded kernel modules.

During the configuration phase all the modules listed in the Watchdogs to Configure will be added to the configuration file and probed immediately.
