High-Availability Configuration Overview

This page allows you to review cluster settings before proceeding with the setup. Click the name of the step in the dialog window to return to it.

configure the corosync communication layer;
Nodes
configure cluster nodes, their host names and IP addresses
NTP Client
configure the Network Time Protocol client for time synchronization;
Fencing (STONITH)
configure cluster fencing mechanism;
Watchdog
configure watchdog kernel modules;
HANA Configuration
(optional, for HANA scenarios) configure HANA-specific parameters;
NetWeaver Configuration
(optional, for NW scenarios) configure NetWeaver-specific parameters.
